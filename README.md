

<p align="center">Вывод курса валют</p>


#### Сборка и запуск проекта:
- Сборка проекта `make build vendor`
- Запуск `make up`
- Перейти на [localhost/](http://localhost/)

#### API:
- ``/api/currency`` - вывод курса валют
- ``api/refresh-currency`` - обновление курса валют

#### Окружение:
- Symfony 7.2.2
- PHP 8.3
- Tailwind
- MySQL
- Nginx
- Redis
- Docker
